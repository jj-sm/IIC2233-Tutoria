# Actividad 2 — Networking TCP + JSON

## Enunciado

**Implementar:**
- Un servidor TCP que espera conexiones.
- Un cliente TCP que se conecta al servidor.
- El cliente envía un diccionario serializado como JSON.
- El servidor recibe el mensaje, lo imprime, y responde "recibido".
- El cliente imprime la respuesta.
- Mantener la conexión abierta y permitir varios envíos.
